# 科学有效性等级与门槛

- `概念草稿`：只有研究想法，或数据适配仍未解决。
- `模拟演示`：使用随机或合成真值，只能验证流程，不能支持领域结论。
- `初步实验`：使用真实任务字段完成了一次受审计实验。
- `可复现研究`：包含基线、多随机种子和统计不确定性。
- `投稿候选`：通过投稿硬门槛和逐主张溯源检查。

数据集与课题不匹配时系统会暂停。人工确认只能让流程继续，不会自动提高科学等级。
模拟演示不能生成投稿稿。

投稿稿至少需要真实任务数据、有效模型、一个基线、三个随机种子、统计不确定性、
完整引用和无未解析科学主张。导出的论文工程会携带代码、数据卡、依赖锁、运行日志、
结果文件和哈希索引。
